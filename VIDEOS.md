## 🚀 Cómo usar

1. **Levantar servidor:**
   ```bash
   cd redfingers
   python3 -m http.server 8000
   ```

2. **Abrir:** `http://localhost:8000`

## ✏️ Para agregar/editar videos

Edita el archivo `videos.js` en el método `getVideos()`:

```javascript
{ id: 'VIDEO_ID', title: 'NOMBRE DEL VIDEO' }
```
