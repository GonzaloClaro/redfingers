# Red Fingers

Sitio oficial de Red Fingers (redfingers.red). HTML/CSS/JS estático, servido vía GitHub Pages.

## Estructura
- `index.html` — página principal (hero, about, productions, galería, contacto)
- `hilorojo/` — página del álbum "Hilo Rojo"
- `videos.js` — lista de tracks (Productions) y lógica de UI (nav, galería, loading screen)
- `styles.css` — estilos
- `_headers`, `robots.txt`, `sitemap.xml` — config despliegue/SEO

## Desarrollo local
```
python3 -m http.server 4321
```
